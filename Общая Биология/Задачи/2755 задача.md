## Решение
| **ХАРАКТЕРИСТИКИ**                                                             | **СТРУКТУРЫ ЭМБРИОНА** |
| ------------------------------------------------------------------------------ | ---------------------- |
| А) имеет генетический материал, наполовину совпадающий с набором генов матери  | 2                      |
| Б) защищает зародыш от механических воздействий                                | 4                      |
| В) эндокринная функция - синтезирует хорионический гонадотропин человека (ХГЧ) | 3                      |
| Г) обеспечивает доставку кислорода от организма матери к плоду                 | 1                      |
| Д) канатик, непосредственно соединяющий зародыш с плацентой                    | 1                      |
| Е) является водной средой для развития зародыша                                | 4                      |
1 - пуповина, 2 - зародыш, 3 - хорион, 4 - амниотическая полость.  
  
1 - пуповина, 2 - зародыш, 3 - хорион, ворсинки хориона, 4 - амниотическая полость.  
А - 2; зародыш содержит гаплоидный набор хромосом от каждого родителя, поэтому его генетический материал наполовину совпадает с материнским;  
Б - 4; амниотическая полость с жидкостью защищает зародыш от ударов и механических повреждений;  
В - 3; хорион синтезирует гормон ХГЧ, который поддерживает беременность на ранних сроках;  
Г - 1; пуповина содержит кровеносные сосуды, через которые осуществляется газообмен между матерью и плодом;  
Д - 1; пуповина представляет собой канатик с сосудами, соединяющий плод с плацентой;  
Е - 4; амниотическая жидкость создает оптимальную водную среду для развития зародыша.

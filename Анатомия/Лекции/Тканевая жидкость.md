- Окружает клетки тела — образуется из жидкой части крови — плазмы, проникающей через стенки кровеносных сосудов в межклеточное пространство. Между тканевой жидкостью и кровью происходит [обмен веществ](https://ru.wikipedia.org/wiki/%D0%9E%D0%B1%D0%BC%D0%B5%D0%BD_%D0%B2%D0%B5%D1%89%D0%B5%D1%81%D1%82%D0%B2 "Обмен веществ"). Часть тканевой жидкости поступает в лимфатические сосуды, образуется лимфа, которая движется по лимфатическим сосудам. По ходу лимфатических сосудов находятся лимфатические узлы, которые играют роль фильтра. Из лимфатических сосудов лимфа изливается в вены, то есть возвращается в кровяное русло.
